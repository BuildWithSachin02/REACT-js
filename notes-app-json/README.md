# 📝 Notes App

A modern and responsive **Notes Application** built with **React**, **Axios**, **Bootstrap**, and **JSON Server**. Users can create personalized notes with custom colors, images, and timestamps. The application supports full CRUD operations and stores data locally using JSON Server.

---

## 🚀 Features

* ✅ Create new notes
* 🎨 Choose custom background and text colors
* 🎲 Automatically assign a random background color if none is selected
* 🖼️ Add images using image URLs
* 📅 Display the note creation date
* 🕒 Display the note creation time
* 🗑️ Delete notes
* ⚡ Real-time data fetching using Axios
* 📱 Responsive UI with Bootstrap
* 🔄 Local REST API using JSON Server

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/BuildWithSachin02/your-repository-name.git
```

### Navigate to the project

```bash
cd your-repository-name
```

### Install dependencies

```bash
npm install
```

---

## ▶️ Run the React Application

```bash
npm run dev
```

---

## ▶️ Start JSON Server

```bash
npx json-server --watch db.json --port 3000
```
