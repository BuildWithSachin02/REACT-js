# 📚 Book Library Management System

A modern Book Library Management System built using **React.js**, **Redux Toolkit**, **JSON Server**, and **Bootstrap**. This application allows users to manage books efficiently with complete CRUD functionality.

---

## 🚀 Features

- 📖 View all books
- ➕ Add new books
- ✏️ Edit existing book details
- 🗑️ Delete books with SweetAlert2 confirmation
- 🔍 View detailed information about each book
- 🔐 User Authentication (Sign Up & Sign In)
- 🛡️ Protected Routes
- 📱 Responsive Design
- ⚡ State Management using Redux Toolkit
- 🌐 REST API integration using JSON Server

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Bootstrap 5
- Axios
- JSON Server
- SweetAlert2

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/BuildWithSachin02/REACT-js.git
```

Go to the project folder

```bash
cd REACT-js/book-library-rtk
```

Install dependencies

```bash
npm install
```

Start React

```bash
npm run dev
```

Start JSON Server

```bash
npx json-server --watch src/json/books.json --port 3000
```
